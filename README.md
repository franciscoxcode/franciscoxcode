## Selected work

**NLP Service for UNAM researchers**
Backend application for word frequency analysis and automatic sentence classification using OpenAI GPT. Researchers define their own classification categories through dynamic prompts and structured JSON responses. Async processing, REST API, Docker deployment.

**Proyecto Angelus · CIDE / Conahcyt / CNB**
Research platform to document forced disappearances during Mexico's Dirty War. Processes thousands of digitized government documents, extracts structured data about missing persons, and presents it as an interactive knowledge graph. Built with Python, Vue.js, PostgreSQL, REST APIs, and Docker.
